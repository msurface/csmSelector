CSS Selector Challenge!

Below you will find the button "Find This Element's CSS Selector" that your client wants to set up a 'Click' goal on.

Here's the catch, the CSS selector tool in Decibel's Portal returns a selector that is to long. This means you will need to find the appropriate CSS Selector for the submit button using Developer Tools.

Good Luck!
