CSS Selector Challenge!

Below you will find the button "Find This Element's CSS Selector" that your client wants to set up a 'Click' goal on.

Here's the catch, the CSS selector tool in Decibel's Portal returns a selector that is to long. This means you will need to find the appropriate CSS Selector for the submit button using Developer Tools.

Good Luck!

CSS Selectors Overview
  • ID Selector

    ID's look like -- id="value"

    #'s are used as ID selectors

    Example -- #value

  • Class Selectors

    Classes look like -- class="value"

    .'s are used as Class selectors

    Example -- .value

  • Element Selector

    Elments's look like -- <button>

    Name's of elements are used as element selectors

    Example -- button

Follow the Hints

  • Steps to help out

  • Step 1: Open Dev Tools.

    Hint: right click on the element and choose "inspect".
  • Step 2: Check for Classes and IDs in Dev Tools on the element.
  • Step 3: Find out if the element has a "Parent" element.
  • Step 4: Test the selector in the Developers Console by running the below code.

    Copy/Paste document.querySelectorAll('CSS Selector Goes Here') into the developers console & swap out the red text with your selector.

    Press enter and view the results
  • Hint: A CSS Combinator will be useful!
